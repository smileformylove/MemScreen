# MemScreen Logo Guidelines

## Logo Assets

MemScreen provides multiple logo sizes for different use cases:

| File | Size | Use Case |
|------|------|----------|
| `logo.png` | 280x280px | Main logo, hero sections, presentations |
| `logo_medium.png` | 120x120px | Section headers, documentation |
| `logo_small.png` | 60x60px | Icons, inline elements, favicons |
| `favicon.png` | 32x32px | Browser favicons, small icons |

## Usage Guidelines

### Do's ✅
- Use the appropriate size for your use case
- Maintain aspect ratio (all logos are square)
- Use on light or dark backgrounds
- Include alt text for accessibility

### Don'ts ❌
- Don't stretch or distort the logo
- Don't change the logo colors
- Don't rotate or flip the logo
- Don't add drop shadows or effects

## Color Palette

The MemScreen brand uses these colors:

```
Primary Green:  #06D6A0  (AI Local)
Secondary Blue: #457B9D  (Privacy)
Accent Red:     #EF476F  (Recording)
Dark Navy:      #1D3557  (Text)
Off-White:      #F1FAEE  (Background)
```
