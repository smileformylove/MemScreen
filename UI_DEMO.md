# MemScreen 统一界面 - 启动指南

## ✅ 界面已成功启动！

UI 进程正在运行中（PID: 18932）。

## 🎨 界面特性

### 四个主标签页

1. **💬 Chat（聊天）**
   - 与 AI 对话，询问关于屏幕内容的问题
   - 选择不同的 Ollama 模型
   - 实时流式响应
   - 自动集成屏幕记忆上下文

2. **🎬 Videos（视频）**
   - 浏览所有屏幕录制
   - 播放视频
   - 时间轴拖动
   - 删除不需要的录制

3. **🔍 Search（搜索）**
   - 搜索屏幕内容
   - 语义搜索
   - OCR 文本搜索

4. **⚙️ Settings（设置）**
   - 查看配置的 AI 模型
   - 数据库位置
   - 使用统计

## 🎯 快速测试

### 测试聊天功能

1. 点击 **Chat** 标签
2. 在输入框中输入问题，例如：
   - "我昨天做了什么？"
   - "帮我找一下关于 Python 的内容"
3. 点击 **Send** 或按 Enter
4. 等待 AI 响应

### 测试视频播放

1. 先录制一些屏幕内容：
   ```bash
   # 在另一个终端运行
   memscreen --duration 10
   ```

2. 点击 **Videos** 标签
3. 从列表中选择一个视频
4. 点击 **Play** 按钮
5. 拖动时间轴查看不同时刻

### 测试搜索功能

1. 点击 **Search** 标签
2. 输入关键词，例如："Python"、"代码"、"设置"
3. 点击 **Search** 按钮
4. 查看搜索结果

## 🔧 故障排除

### 界面没有显示
- 检查进程是否运行：`ps aux | grep memscreen`
- 重启界面：`python3 -m memscreen.unified_ui`

### 聊天没有响应
- 确保 Ollama 正在运行：`ollama serve`
- 检查模型是否已下载：`ollama list`

### 视频无法播放
- 确保已经录制过屏幕：`memscreen --duration 10`
- 检查数据库是否存在：`ls ./db/screen_capture.db`
